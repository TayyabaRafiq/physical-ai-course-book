# Contributors

## Project Team

This project was created as part of the AI Textbook First initiative to demonstrate practical integration of Large Language Models with robotics systems.

## Acknowledgments

### Core Technologies

- **OpenAI Whisper** - Speech recognition model enabling voice command processing
- **OpenAI GPT-4** - Large language model for intent parsing and action planning
- **LangChain/LangGraph** - Agentic framework for cognitive planning workflows
- **Pydantic** - Data validation and settings management
- **Click** - CLI framework for user interaction
- **PyAudio** - Audio capture interface
- **Structlog** - Structured logging system

### Optional Extensions

- **ROS 2** - Robot Operating System (optional backend for real robot control)
- **Gazebo** - Physics simulation (optional for hardware-in-the-loop testing)
- **PyTorch** - Deep learning framework (for Whisper GPU acceleration)

## Contributing

This is an educational research prototype demonstrating voice-to-action pipelines for humanoid robots.

For questions, issues, or contributions, please:
1. Review the documentation in `docs/` and `specs/`
2. Check existing issues and discussions
3. Submit detailed bug reports or feature requests
4. Include test cases and examples with pull requests
