## Steps for Creating Docusaurus Sidebar Configuration

1.  **Create `sidebars.js`**: If it doesn't exist, create a `sidebars.js` file in the Docusaurus project root (usually alongside `docusaurus.config.js`).
2.  **Define Sidebar Item**: Within `sidebars.js`, define an object for the "Robot Nervous System (ROS 2)" module.
3.  **Specify `type: 'autogenerated'`**: For simplicity and to automatically pick up the `docs/01-robot-nervous-system/` content, set the `type` to `autogenerated` and point `dirName` to `01-robot-nervous-system`.
4.  **Add to `module.exports`**: Export the defined sidebar item as part of `module.exports`.

Example `sidebars.js` (conceptual, assuming project root):

```javascript
// sidebars.js
/** @type {import('@docusaurus/plugin-content-docs').SidebarsConfig} */
const sidebars = {
  tutorialSidebar: [
    {
      type: 'autogenerated',
      dirName: '01-robot-nervous-system', // Generate sidebar from the docs/01-robot-nervous-system folder
    },
  ],
};

module.exports = sidebars;
```
